## To-Do Web Application
This is a simple and interactive To-Do Web Application developed using HTML, CSS, and JavaScript. The application helps users manage their daily tasks efficiently by allowing them to add, edit, delete, and mark tasks as completed.
## Technologies Used
* HTML – Structure of the application
* CSS – Styling and layout
* JavaScript – Functionality and logic
* LocalStorage – Data persistence
